Classification

A classifier is a supervised learning model for categorical targets.
Instead of predicting a continuous value, we model:

P(Y=y∣X1=x1,…,Xp=xp)P(Y=y \mid X_1=x_1,\dots,X_p=x_p)P(Y=y∣X1​=x1​,…,Xp​=xp​)

Direct construction (categorical predictors)

If all predictors are categorical, we can use:

P(Y=y∣X1=x1,…,Xp=xp)=P(Y=y,X1=x1,…,Xp=xp)P(X1=x1,…,Xp=xp)P(Y=y\mid X_1=x_1,\dots,X_p=x_p) = \frac{P(Y=y,X_1=x_1,\dots,X_p=x_p)} {P(X_1=x_1,\dots,X_p=x_p)}P(Y=y∣X1​=x1​,…,Xp​=xp​)=P(X1​=x1​,…,Xp​=xp​)P(Y=y,X1​=x1​,…,Xp​=xp​)​

with

P(X1=x1,…,Xp=xp)=∑yP(Y=y,X1=x1,…,Xp=xp)P(X_1=x_1,\dots,X_p=x_p)=\sum_y P(Y=y,X_1=x_1,\dots,X_p=x_p)P(X1​=x1​,…,Xp​=xp​)=y∑​P(Y=y,X1​=x1​,…,Xp​=xp​)

In practice we estimate these probabilities by empirical proportions.

Limitation of direct counting

If predictors are binary, the number of joint probabilities grows as 2p+12^{p+1}2p+1.
This becomes too large quickly, so we need direct conditional models.

Logistic Regression

For binary target Y∈{0,1}Y\in\{0,1\}Y∈{0,1}, define linear predictor:

ηi=β0+∑j=1pβjxi,j\eta_i=\beta_0+\sum_{j=1}^p \beta_j x_{i,j}ηi​=β0​+j=1∑p​βj​xi,j​

Map ηi\eta_iηi​ to probability with logistic function:

P(Yi=1∣xi)=11+e−ηiP(Y_i=1\mid x_i)=\frac{1}{1+e^{-\eta_i}}P(Yi​=1∣xi​)=1+e−ηi​1​

Odds and log-odds

Odds in favor of class 1:

Oi=P(Yi=1∣xi)P(Yi=0∣xi)O_i=\frac{P(Y_i=1\mid x_i)}{P(Y_i=0\mid x_i)}Oi​=P(Yi​=0∣xi​)P(Yi​=1∣xi​)​

Logistic regression is equivalent to:

log⁡Oi=log⁡P(Yi=1∣xi)P(Yi=0∣xi)=β0+∑j=1pβjxi,j\log O_i=\log\frac{P(Y_i=1\mid x_i)}{P(Y_i=0\mid x_i)} =\beta_0+\sum_{j=1}^p\beta_j x_{i,j}logOi​=logP(Yi​=0∣xi​)P(Yi​=1∣xi​)​=β0​+j=1∑p​βj​xi,j​

Interpretation:

  • β0\beta_0β0​: log-odds when all predictors are 0.
  • βj\beta_jβj​: change in log-odds per one-unit increase in xjx_jxj​.

Estimating Logistic Regression (MLE)

Let

θi(β0,β)=11+exp⁡ ⁣(−β0−∑j=1pβjxi,j)\theta_i(\beta_0,\beta)=\frac{1}{1+\exp\!\left(-\beta_0-\sum_{j=1}^p\beta_j x_{i,j}\right)}θi​(β0​,β)=1+exp(−β0​−∑j=1p​βj​xi,j​)1​

For new feature vector x′x'x′:

  • predict class 1 if P^(Y′=1∣x′)>0.5\hat P(Y'=1\mid x')>0.5P^(Y′=1∣x′)>0.5 (equivalently η^>0\hat\eta>0η^​>0),

Evaluating Classifiers

Confusion matrix terms

  • TP: predicted 1, true 1
  • TN: predicted 0, true 0
  • FP: predicted 1, true 0
  • FN: predicted 0, true 1

Metrics

Classification accuracy:

CA=TP+TNTP+TN+FP+FNCA=\frac{TP+TN}{TP+TN+FP+FN}CA=TP+TN+FP+FNTP+TN​

Sensitivity (true positive rate):

TPR=TPTP+FNTPR=\frac{TP}{TP+FN}TPR=TP+FNTP​

Specificity (true negative rate):

TNR=TNTN+FPTNR=\frac{TN}{TN+FP}TNR=TN+FPTN​

ROC and AUC

The model outputs a predicted score, such as P(Y=1∣X)P(Y=1 \mid X)P(Y=1∣X).
We choose a threshold TTT to convert the score into a class prediction.

Y^={1,if score≥T0,if score<T\hat{Y} = \begin{cases} 1, & \text{if score} \ge T \\ 0, & \text{if score} < T \end{cases}Y^={1,0,​if score≥Tif score<T​

By changing the threshold (T), we get different values of sensitivity and specificity.

sensitivity=TPTP+FN\text{sensitivity} = \frac{TP}{TP+FN}sensitivity=TP+FNTP​ specificity=TNTN+FP\text{specificity} = \frac{TN}{TN+FP}specificity=TN+FPTN​

The ROC curve plots:

sensitivityagainst1−specificity (FPR)\text{sensitivity} \quad \text{against} \quad 1 - \text{specificity}\text{ (FPR)}sensitivityagainst1−specificity (FPR)

where 1−specificity1 - \text{specificity}1−specificity is the false positive rate (FPR).

AUC is the area under the ROC curve.

AUC can be interpreted as the probability that a randomly chosen class-1 sample receives a higher predicted score than a randomly chosen class-0 sample.

For example, if:

AUC=0.8AUC = 0.8AUC=0.8

then the model has an 80% chance of ranking a random positive sample higher than a random negative sample.

In simple terms, AUC measures how well the model separates class 1 from class 0 across all possible thresholds.
